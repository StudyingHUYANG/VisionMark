# 本地开发配置说明

## 📝 配置文件位置

`extension/content/config.local.js`

## 🔧 如何修改端口

1. 打开 `extension/content/config.local.js`
2. 修改 `API_BASE` 的值：
   ```javascript
   window.LOCAL_CONFIG = {
     // 修改这里！
     API_BASE: 'http://localhost:你的端口',
     API_VERSION: 'api/v1'
   };
   ```

## 🌍 常见配置示例

### 本地开发（默认）
```javascript
API_BASE: 'http://localhost:8080'
```

### 远程生产服务器
```javascript
API_BASE: 'https://apitest.visionmark.com.cn:8443'
```

### 其他端口
```javascript
API_BASE: 'http://localhost:3000'  // 或 5000, 8000 等
```

## ⚠️ 注意事项

- **不要提交** `config.local.js` 到 Git（已加入 .gitignore）
- 每个开发者有自己的配置副本
- 修改配置后需要刷新插件才能生效

## 🔄 修改后如何生效

1. 保存 `config.local.js`
2. 打开 `chrome://extensions/`
3. 点击插件的"刷新"按钮 🔄
4. 刷新 B 站视频页面
